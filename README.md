Reimagine Renovations - Upgraded Vite + React + Tailwind Project (Bright Hero)

Features:
- Splash screen (2s) with logo
- Sticky responsive navbar with hamburger menu
- Full-screen hero with bright clean interior background (temporary Unsplash image)
- Smooth scroll and anchor navigation
- Services grid, Gallery placeholders, Contact form (Formspree)
- SEO meta + favicon + _redirects included

Local dev:
1. npm install
2. npm run dev (open http://localhost:5173)

Build:
1. npm run build => outputs /dist

Deploy to Cloudflare Pages:
- Framework preset: React (Vite)
- Build command: npm run build
- Output directory: dist

Replace /public/logo.png and /public/favicon.ico with your files.
